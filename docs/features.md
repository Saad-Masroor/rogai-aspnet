# ROGAI Features

## Version 1 (Core)
- Player registration & login
- Dashboard showing current run stats
- Start a new run
- Track player stats: health, gold, inventory
- Procedural event generator (random encounters)
- Death handling: restart run
- Leaderboard for longest survival / high score

## Version 2 (Intermediate)
- Multiple AI types or classes
- Items & equipment system
- Persistent achievements
- Enhanced procedural events

## Version 3 (Advanced)
- REST API for frontend separation
- Optional graphical frontend using Canvas or Phaser
- Real-time multiplayer interactions
