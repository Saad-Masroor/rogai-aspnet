# ROGAI Development Roadmap

**Phase 1 – Setup & Core Systems**
1. Create ASP.NET Core Razor Pages project
2. Setup database connection (SQL Server)
3. Create Users table and model
4. Implement user registration
5. Implement user login
6. Create “Start New Run” logic
7. Track health, gold, and inventory
8. Implement procedural event generator
9. Handle death and restart run
10. Build dashboard displaying current run stats
11. Implement leaderboard for high scores

**Phase 2 – Intermediate Features**
12. Add AI types/classes
13. Add items & equipment
14. Track persistent achievements
15. Expand procedural events logic

**Phase 3 – Advanced Features**
16. Create REST API for frontend separation
17. Add optional graphics / animations
18. Implement real-time multiplayer (SignalR/WebSockets)
19. Optimize database and backend for scale
20. Polish UI and document system fully
