# ROGAI Vision

ROGAI (Rogue AI) is a browser-based roguelike strategy game where each run is unique. If the player dies, the game restarts. The focus is on **backend systems, procedural gameplay, and database management**, not graphics—though graphics can be added later.

Goals:
- Learn and demonstrate ASP.NET Core backend development
- Learn SQL Server and database design
- Track player progress and high scores
- Implement procedural events and run logic
- Showcase professional version control and documentation practices
